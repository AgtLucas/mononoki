the mononoki project started with the search for a better programming font. I tried every suitable monospace typeface i could get my hands on, but i couldn't find the optimal solution, at least not for me. So it slowly became a small, personal research-project, exploring the typographic needs of code display. I used the following specifications as guidelines:
# fizzbuzz! for i in xrange(1, 101): if i % 15 == 0: print "FizzBuzz" elif i % 3 == 0: print "Fizz" elif i % 5 == 0: print "Buzz" else: print i
